# X Card Generator / X 名片生成器

输入任意 X（Twitter）主页链接，一键生成精美名片 & 二维码。

## ✨ 功能

- 🔗 智能解析 URL（支持 `x.com/user` / `@username` / `twitter.com/...`）
- 📱 生成高清二维码，扫码直达主页
- 🎨 精美 X 风格名片卡片（带头像、昵称、简介）
- 🌙 暗黑模式优先
- 📱 移动端完美适配
- ✏️ 名片信息实时编辑
- 📥 下载二维码 PNG / 下载整张名片图片
- 📋 一键复制主页链接

## 🚀 使用

直接在浏览器打开 `twitter-card/index.html` 即可，无需安装任何依赖。

## 📷 预览

1. 输入 X 主页 URL → 点击"生成名片"
2. 名片即时生成，头像自动拉取
3. 编辑名称/简介，名片实时更新
4. 下载二维码或整张名片图片

## 🛠 技术栈

- HTML5 + Vanilla JS
- Tailwind CSS（CDN）
- goQR API（二维码生成）
- unavatar.io（头像获取）
- html2canvas（名片截图下载）
